# Contributing to Leafy Talks

Thank you for considering contributing to Leafy Talks! Here’s how you can help:

## Ways to Contribute
- **Documentation:** Improve existing documentation or add new guides.
- **Code:** Write code to add new features or fix bugs.
- **Testing:** Help us by writing tests and finding bugs.
- **Feedback:** Provide suggestions or constructive criticism.

## Guidelines
- Fork the repository.
- Create a feature branch (e.g., `feature/my-feature`).
- Make your changes.
- Ensure tests pass.
- Submit a Pull Request.

### Code of Conduct
Please adhere to our [Code of Conduct](./CODE_OF_CONDUCT.md).

Thank you for improving Leafy Talks!
